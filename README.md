Project Overview

Engineered a responsive Clash of Clans fan site featuring a dynamic theming engine that allows real-time UI/UX customization based on iconic in-game characters such as Hog Rider, Archer Queen, and Village Girl. Themes—including color schemes, hero assets, and animations—transition seamlessly without requiring page reloads, built entirely with native web technologies.

Key Contributions

🔧 Scalable Theming Architecture

Developed a maintainable CSS variable system with 25+ custom properties per theme.

Enabled real-time theme switching using document.documentElement.style and classList without any external libraries.

Achieved smooth transitions through hardware-accelerated CSS keyframes and transform animations.

📁 State Management & Performance

Integrated localStorage to persist user theme preferences across sessions, reducing repeat asset requests.

Implemented runtime asset switching via dynamic url() injection for themed backgrounds and SVG elements.

Delivered near-instant theme changes, maintaining a fluid user experience across all devices.

🎮 Enhanced Interactivity

Created a character comparison dashboard with visually organized stats (e.g., win/loss ratio, trophies).

Implemented persona-specific micro-interactions—such as animated bounce effects for Goblins—to deepen immersion.

Outcome

The application remained fast and responsive across screen sizes, utilizing modern CSS layout and typography techniques.
